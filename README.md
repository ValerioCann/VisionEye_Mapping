# VisionEye_Mapping

Vision Eye Mapping with Ultralytic's YOLOv8 nano & Open CV

The Notebook contains codes for VisionEye Mapping with distance calculation from the origin point and Pinpoint Mapping

### Librairies
- Ultralytic
- Open CV
